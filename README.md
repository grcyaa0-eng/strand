# Strand

AI-powered ingredient analyzer for textured hair. Scan a product label, get personalized analysis based on your hair profile.

## Tech Stack

- React + Vite + TypeScript
- Tailwind CSS
- Supabase (auth + database)
- Claude API (vision + chat)
- Deployed on Vercel

## Status

🚧 In active development — Summer 2026

## Getting Started

```bash
npm install
npm run dev
```
